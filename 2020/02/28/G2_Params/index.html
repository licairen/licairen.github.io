高度、宽度以及内边距

1.高度

所有图表均可通过:height=""配置高度；

属性 说明 类型 可选值 默认值
radius 环半径 `String Number` `’50%’
activeRadius 环半径（动态） `String Number` `’55%’
data 环数据 Array data属性 []
lineWidth 环线条宽度 Number — 20
activeTimeGap 切换间隔(ms) Number — 3000
color 环颜色 Array [1] []
digitalFlopStyle 数字翻牌器样式 Object — [2]
digitalFlopToFixed 数字翻牌器小数位数 Number — 0
animationCurve 动效曲线 String Transition 'easeOutCubic'
animationFrame 动效帧数 Number [3] 50

2.宽度

词云图通过:width=""来配置宽度，除此之外的图表通过style="width: px;"配置宽度；

3.内边距

迷你图的内边距默认为0，除此之外的图表的通过:padding="[/'auto', /'auto']配置内边距;

坐标轴颜色

1.颜色说明

支持字符串、16进制、rgb、rgba等形式的数值

2.通用调整方案

适用于:

  • area 面积图
  • bubble 气泡图
  • column 柱状图/条形图
  • double-axis-column 双Y轴柱状图/条形图 (三个轴，x,y1,y2)
  • line 折线图
  • scatter-point 散点图

3.需要特殊调整

适用于:

  • histogram 直方图

适用于:

  • mirror-interval 镜像分面柱图

适用于:

  • liquidfill 液体填充

适用于:

  • progress-bar 进度条

Tooptip
